TransRectal Ultrasound Test (TRUS)

  • In this exam a small lubricated probe is inserted into the rectum. The probe has sound waves that bounce off the prostate. This produces an image that the doctor can see on a video screen.
  • You will feel some pressure when the probe is inserted, but it is usually not painful. A local anesthetic can be used to reduce any pain.
Rectum

Rectum: The lower part (last 8-10 inches) of the large intestine. The rectum stores solid waste (stool) until it leaves the body thorough the anus.
